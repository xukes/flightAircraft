# 音效文件说明

这个文件夹包含了游戏所需的所有音效文件。

## 需要的音效文件列表：

### 武器音效
- `shoot.wav` - 玩家普通射击音效
- `laser.wav` - 激光武器启动和持续音效
- `enemy_shoot.wav` - 敌人射击音效

### 互动音效
- `hit.wav` - 击中敌人或受到伤害的音效
- `explosion.wav` - 敌人爆炸音效
- `powerup.wav` - 收集道具音效

### 特殊效果
- `lightning.wav` - 闪电攻击音效
- `game_over.wav` - 游戏结束音效

### 背景音乐
- `background_music.mp3` - 游戏背景音乐（循环播放）

## 音效文件要求：

### 格式
- **推荐格式**: WAV (音效) / MP3 (音乐)
- **采样率**: 44.1kHz
- **位深度**: 16-bit

### 音效长度建议
- 射击音效: 0.1-0.3秒
- 爆炸音效: 0.5-1.0秒
- 道具音效: 0.2-0.5秒
- 闪电音效: 1.0-2.0秒
- 游戏结束音效: 2.0-3.0秒
- 背景音乐: 循环，建议1-2分钟

### 音量建议
- 确保音效文件本身音量适中，不要过载
- 背景音乐音量应该比音效低一些

## 获取音效资源：

### 免费资源网站
1. **Freesound.org** - 大量免费音效，需注意许可证
2. **Zapsplat** - 免费注册后可下载
3. **Bfxr.net** - 在线生成8bit风格音效
4. **sfxr.me** - 另一个在线音效生成器

### 推荐的音效风格
- **8bit/复古风格** - 适合像素风格游戏
- **科幻风格** - 激光、爆炸等特效
- **电子音乐** - 背景音乐

### 使用音效生成器
对于射击游戏，推荐使用 Bfxr 或类似的音效生成器：

1. 访问 https://www.bfxr.net/
2. 点击 "Randomize" 生成基础音效
3. 调整参数获得想要的效果：
   - **射击音效**: 调整 Attack Decay 和 Sustain
   - **爆炸音效**: 增加 Low Pass 和 Bit Crush
   - **道具音效**: 提高 Pitch 和使用 Vibrato

### 音效编辑软件
- **Audacity** (免费) - 基础编辑和效果处理
- **LMMS** (免费) - 音乐制作
- **Reaper** (付费) - 专业音频编辑
