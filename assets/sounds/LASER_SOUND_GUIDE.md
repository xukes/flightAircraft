# 🔫 激光音效详细指南

## 问题描述
激光武器缺乏完整的音效反馈，包括：
- 激光启动音效
- 激光持续音效
- 激光击中敌人音效

## 解决方案

### 1. 激光启动音效 (laser.wav)
**特点**：
- 短促、有力的启动声
- 适合作为激光武器激活的提示
- 音调应该比普通射击高一些

**制作方法 (Bfxr)**：
1. 点击 "Laser" 预设
2. 增加 sustain 时间到 0.2-0.3
3. 提高 pitch 让声音更尖锐
4. 添加轻微的 vibrato

### 2. 激光持续音效 (laser_loop.wav) - 可选
如果想让激光有持续的音效，可以创建一个循环版本：
**特点**：
- 更低的音量 (30%)
- 稍低的音调 (rate: 0.8)
- 可以无缝循环
- 适合长时间激光效果

### 3. 激光击中音效 (hit.wav 的变种)
**代码中的实现**：
```typescript
// 激光击中时使用特殊的音效参数
this.audioManager.playSound('hit', {
    volume: 0.3,
    rate: 1.5 // 更高的音调
});
```

**效果**：
- 激光击中敌人时播放
- 音量较低 (30%)，避免盖过其他音效
- 更高的音调 (rate: 1.5)，听起来更像是激光灼烧
- 使用现有的 hit.wav 文件，但通过参数调整获得不同效果

## 文件需求

### 必需文件
- `laser.wav` - 激光启动音效
- `hit.wav` - 基础击中音效

### 可选文件
- `laser_loop.wav` - 激光持续循环音效

## 音效调试

### 如果激光没有声音：
1. **检查文件**：确认 `laser.wav` 文件存在于 `assets/sounds/` 目录
2. **检查文件名**：确保文件名完全匹配（区分大小写）
3. **检查格式**：确保是有效的音频格式
4. **浏览器控制台**：查看是否有音频加载错误

### 音量调整
在 `MainScene.ts` 中可以调整激光音效的音量：
```typescript
this.laserSound = this.sound.add('laser', { volume: 0.5 }); // 50% 音量
this.laserLoopSound = this.sound.add('laser', { volume: 0.3 }); // 30% 音量
```

### 替代方案
如果没有专门的激光音效，可以使用：
- 修改 `shoot.wav` 的音调
- 使用电子合成器生成类似效果
- 在线搜索 "laser sound effect free"

## 测试步骤

1. **基础测试**：启动游戏，按空格键激活激光
2. **持续测试**：按住空格键，激光应该有启动音效
3. **击中测试**：用激光击中敌人，应该有特殊的击中音效
4. **结束测试**：松开空格键，激光音效应该停止
